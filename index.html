Komponen 01

Motherboard

Main Board / System Board

Papan sirkuit utama yang menjadi tulang punggung komputer — menghubungkan semua komponen seperti CPU, RAM, storage, dan periferal melalui jalur bus, slot, dan konektor yang terintegrasi.

Fungsi Utama

Menghubungkan seluruh komponen komputer melalui sistem bus. Menyediakan socket prosesor, slot memori, slot ekspansi (PCIe, PCI), konektor storage (SATA, NVMe), dan port I/O. Mendistribusikan daya dari PSU ke semua komponen.

Kerusakan Umum

Korsleting akibat tegangan tidak stabil, kapasitor bengkak atau bocor, jalur PCB putus, komputer tidak mau POST, port USB/audio rusak, kerusakan akibat lonjakan listrik (petir/power surge).

Solusi

Gunakan UPS/stabilizer, cek kapasitor secara berkala (visual), bersihkan debu secara rutin, pastikan konektor terpasang sempurna, ganti baterai CMOS jika sistem tidak menyimpan konfigurasi.

Komponen 02

Processor

CPU — Central Processing Unit

Otak dari sistem komputer. Menjalankan instruksi program, melakukan operasi aritmetika, logika, kontrol, dan I/O melalui siklus fetch-decode-execute dalam miliaran siklus per detik.

Fungsi Utama

Mengeksekusi instruksi program. Komponen utama: ALU (Arithmetic Logic Unit), CU (Control Unit), dan register. Berkomunikasi dengan RAM melalui Northbridge/IMC, mengontrol seluruh alur data di sistem.

Kerusakan Umum

Overheat akibat pasta termal kering, pin socket bengkok (PGA), BSOD, komputer restart sendiri, kinerja menurun drastis, CPU tidak terdeteksi di BIOS, kerusakan akibat ESD (electrostatic discharge).

Solusi

Ganti pasta termal setiap 2–3 tahun, pastikan heatsink terpasang kencang, gunakan gelang anti-static saat pemasangan, monitor suhu CPU (ideal <80°C under load), pastikan socket bersih dari debu.

Komponen 03

RAM

Random Access Memory — Main Memory

Memori kerja komputer yang menyimpan data dan instruksi yang sedang diproses secara aktif. Bersifat volatile — data hilang saat tidak ada daya. Kecepatan dan kapasitasnya langsung mempengaruhi performa multitasking.

Fungsi Utama

Menyediakan ruang kerja cepat bagi CPU dan OS. Menyimpan data program yang aktif, buffer I/O, dan cache OS. Lebih cepat dari storage namun lebih lambat dari cache L1/L2/L3. Jenis: DIMM (desktop), SO-DIMM (laptop).

Kerusakan Umum

BSOD berulang, komputer sering freeze, gagal POST dengan bunyi beep, sistem tidak mengenali kapasitas penuh, performa menurun, blue screen dengan kode memory management, artefak visual pada saat booting.

Solusi

Jalankan Windows Memory Diagnostic atau MemTest86, coba pasang satu modul sekaligus, bersihkan slot dan konektor modul dengan eraser/penghapus, pastikan RAM kompatibel dengan motherboard (cek QVL), update BIOS.

Komponen 04

ROM / Storage

Read-Only Memory / Secondary Storage

Penyimpanan non-volatile yang menyimpan firmware sistem (BIOS/UEFI) serta data permanen pengguna. Mencakup ROM chip (firmware), HDD, SSD, dan NVMe sebagai penyimpanan sekunder.

Fungsi Utama

ROM chip menyimpan firmware BIOS/UEFI yang dijalankan saat booting. Storage sekunder (HDD/SSD) menyimpan OS, aplikasi, dan data pengguna secara permanen. Dihubungkan melalui IDE, SATA, atau NVMe (PCIe).

Kerusakan Umum

HDD: bad sector, suara klik, head crash. SSD: sel NAND habis (write endurance), tidak terdeteksi. ROM/BIOS: gagal flash, brick. IDE/SATA: kabel rusak, konektor longgar, tidak terdeteksi di BIOS/UEFI.

Solusi

Monitor kesehatan HDD/SSD dengan CrystalDiskInfo (pantau SMART), backup berkala, jangan cabut storage saat sedang aktif, hindari getaran fisik pada HDD, update firmware SSD, gunakan flash BIOS dengan hati-hati sesuai panduan vendor.

Komponen 05

BIOS & CMOS Battery

Basic Input Output System / CMOS Battery

BIOS adalah firmware yang dijalankan pertama kali saat komputer menyala — melakukan POST dan menginisialisasi hardware. Baterai CMOS menjaga konfigurasi BIOS dan jam real-time (RTC) tetap aktif saat komputer mati.

Fungsi Utama

BIOS/UEFI: menjalankan POST (Power-On Self Test), memuat OS dari bootable device, menyediakan interface antara hardware dan OS. CMOS Battery (CR2032): menyimpan setting BIOS, tanggal/waktu sistem, dan konfigurasi IC lain (sound card, dll).

Kerusakan Umum

Jam sistem selalu reset ke default, muncul pesan "CMOS Checksum Error" atau "Press F2 to enter setup", komputer tidak bisa POST, tanggal kembali ke 2000/1980 setiap restart, setting BIOS tidak tersimpan.

Solusi

Ganti baterai CMOS (CR2032, ~Rp5.000) — umur baterai 3–5 tahun. Jika masih eror setelah diganti, cek chip CMOS di motherboard. Untuk BIOS brick: gunakan BIOS Flashback atau programmer eksternal. Update BIOS hanya saat diperlukan.

Evolusi BIOS → UEFI

Legacy BIOS (1975) BIOS dengan GUI (2000-an) EFI — Intel (2002) UEFI 2.0 (2006) UEFI + Secure Boot (2012+) UEFI + TPM 2.0 (Win11 era)

Komponen 06

Power Supply Unit

PSU — Power Supply Unit

Konverter daya yang mengubah listrik AC dari sumber daya (220V/50Hz) menjadi tegangan DC yang dibutuhkan komponen komputer: +12V, +5V, +3.3V, dan -12V. Kualitas PSU langsung mempengaruhi stabilitas dan umur komponen lain.

Fungsi & Konektor

Mendistribusikan daya ke motherboard (24-pin ATX), CPU (4/8-pin EPS), GPU (6/8-pin PCIe), storage (SATA power), dan periferal (Molex). ATX PSU mendukung soft-power (PS_ON signal), sleep states, dan wake-on-LAN.

Kerusakan Umum

Komputer tidak menyala sama sekali, restart acak di bawah beban tinggi, komponen lain rusak akibat tegangan tidak stabil, bau terbakar, kipas PSU tidak berputar, dan output voltage tidak sesuai spesifikasi.

Solusi

Gunakan PSU dengan sertifikasi 80 PLUS (Bronze/Gold/Platinum), hitung kebutuhan watt sebelum membeli (GPU + CPU + komponen lain + 20% headroom), gunakan UPS/surge protector, ganti PSU jika sudah lebih dari 5–7 tahun.

Komponen 07

Chipset

Northbridge & Southbridge / PCH

Jembatan data antara prosesor dan komponen eksternal. Arsitektur klasik terdiri dari Northbridge (internal: CPU–RAM–GPU) dan Southbridge (eksternal: I/O, USB, SATA). Modern: digabung dalam satu chip PCH (Platform Controller Hub).

Fungsi Northbridge

Mengatur komunikasi antara CPU, RAM (via IMC), dan slot AGP/PCIe. Menentukan tipe, kecepatan, dan jumlah RAM yang dapat digunakan. Berkomunikasi dengan Southbridge melalui internal bus (DMI pada Intel).

Fungsi Southbridge

Mengontrol: bus IDE/SATA, USB, Plug & Play, PCI–ISA bridge, keyboard/mouse controller, power management (ACPI), audio onboard, dan periferal I/O lainnya. Terhubung ke Northbridge melalui bus lebih lambat.

Kerusakan & Solusi

Chipset rusak (overheat/voltase): port USB tidak bekerja, storage tidak terdeteksi, audio mati. Solusi: cek heatsink chipset, tidak bisa diperbaiki sendiri — harus reflow/reball profesional atau ganti motherboard.

Evolusi Arsitektur Chipset

Northbridge + Southbridge (1990-an) Intel 915/945/975 Chipset AMD 790FX / nForce Intel PCH (Cougar Point, 2011) IMC integrasi ke CPU (Sandy Bridge+) AMD Fusion Controller Hub PCH Z790 / X670E (2022–2023)

System-on-Chip (SoC) modern: VGA, audio, dan modem telah on-board dalam satu chip — seperti yang digunakan laptop dan mobile workstation. Contoh chip audio onboard: Realtek ALC888 / ALC897 / ALC4080.
