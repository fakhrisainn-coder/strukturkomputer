Komponen Utama Sistem Komputer


Sistem komputer terdiri dari beberapa bagian utama yang bekerja bersama:


  • Processor / CPU sebagai pusat pemrosesan data.
  • Main Memory sebagai penyimpanan sementara.
  • I/O Modules untuk komunikasi perangkat input dan output.
  • System Interconnection sebagai jalur komunikasi antar komponen.

Motherboard

Product (English): Motherboard / Mainboard

Deskripsi

Motherboard adalah papan utama komputer yang menghubungkan seluruh komponen seperti CPU, RAM, storage, dan perangkat tambahan.

Fungsi

Sebagai pusat koneksi hardware melalui socket processor, slot RAM, chipset, PCI Express, USB, dan konektor power.

Kerusakan Umum

• Tidak menyala
• Komponen tidak terbaca
• BIOS error
• Port rusak

Solusi

Membersihkan motherboard, cek jalur listrik, update BIOS, atau mengganti motherboard jika kerusakan berat.

Perkembangan

Dari motherboard AT lama berkembang menjadi ATX, Micro ATX, Mini ITX dengan dukungan DDR5, PCIe 5.0 dan teknologi chipset terbaru.

RAM

Product (English): Random Access Memory

Deskripsi

RAM adalah memori sementara yang digunakan komputer untuk menjalankan aplikasi dan proses sistem.

Fungsi

Menyimpan data sementara agar processor dapat mengakses data lebih cepat.

Kerusakan Umum

• Blue Screen
• Komputer restart sendiri
• Sistem lambat

Solusi

Membersihkan pin RAM, melakukan test memory, atau mengganti modul RAM.

Perkembangan

Generasi:
SDRAM → DDR → DDR2 → DDR3 → DDR4 → DDR5.

DDR5 memiliki bandwidth tinggi dan kapasitas dapat mencapai lebih dari 128GB per modul.

Processor

Product (English): CPU / Central Processing Unit

Deskripsi

Processor adalah otak komputer yang melakukan perhitungan dan menjalankan instruksi.

Fungsi

Mengolah data, menjalankan aplikasi, dan mengatur kerja seluruh sistem.

Kerusakan Umum

• Overheat
• Komputer mati mendadak
• Performa menurun

Solusi

Membersihkan heatsink, mengganti thermal paste, dan menggunakan sistem pendingin yang baik.

Perkembangan

Dari single core berkembang menjadi multi-core processor seperti Intel Core dan AMD Ryzen dengan kecepatan GHz tinggi.

ROM

Product (English): Read Only Memory

Deskripsi

ROM adalah media penyimpanan permanen yang berisi firmware komputer.

Fungsi

Menyimpan BIOS/UEFI untuk proses booting komputer.

Kerusakan Umum

Firmware rusak, gagal booting, BIOS corrupt.

Solusi

Melakukan flashing BIOS atau mengganti chip firmware.

Perkembangan

Berkembang dari ROM, PROM, EPROM hingga Flash ROM modern.

CMOS Battery

Product (English): CMOS Battery

Deskripsi

Baterai kecil pada motherboard yang menjaga konfigurasi BIOS.

Fungsi

Menyimpan waktu, tanggal, dan pengaturan BIOS ketika komputer mati.

Kerusakan Umum

Jam BIOS berubah, muncul pesan CMOS error.

Solusi

Mengganti baterai CMOS dengan tipe CR2032 baru.

Perkembangan

Saat ini digunakan baterai lithium dengan umur panjang dan konsumsi rendah.

Power Supply

Product (English): Power Supply Unit (PSU)

Deskripsi

PSU mengubah listrik AC menjadi DC untuk komponen komputer.

Fungsi

Memberikan energi stabil kepada motherboard, CPU, GPU dan storage.

Kerusakan Umum

• Komputer mati total
• Tegangan tidak stabil
• Suara kipas abnormal

Solusi

Membersihkan PSU, cek kabel power, atau mengganti PSU berkualitas.

Perkembangan

AT berkembang menjadi ATX dengan sertifikasi efisiensi 80 Plus Bronze hingga Platinum.

Chipset

Product (English): Chipset Controller

Deskripsi

Chipset adalah pengatur komunikasi antara processor, RAM, storage dan perangkat I/O.

Fungsi

Mengontrol jalur data dan menentukan fitur motherboard.

Kerusakan Umum

• USB tidak bekerja
• Hardware tidak terdeteksi
• Sistem tidak stabil

Solusi

Update driver chipset, cek motherboard, atau mengganti board.

Perkembangan

Dahulu menggunakan Northbridge dan Southbridge, sekarang banyak fungsi digabung menjadi satu chipset modern.
